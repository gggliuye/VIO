Capella Localization
==============================

**Capella** Workspace of **Server Localization** process.
`Capella <https://www.eclipse.org/capella/>`_  is a comprehensive, extensible and field-proven MBSE tool and method
to successfully design systems architecture.

The system has the following parts:

* User scan application, and server map building process.
* Our implementation of a local SLAM system.
* Single image based localization system.
* Marker image based camera localization, combined with server cloud marker check.
* etc
